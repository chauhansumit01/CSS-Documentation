  12. positioning

    13. the position of the element will only change if the position property of the element is specified .
  5 types of positioning
  -static -> does not affect positioning will be positioned acc to the document
  -relative -> changes the position with respect to the the original position
  -fixed -> this element position stays fixed to the viewport even if the document is scrolled up or down
  -absolute-> positions the element with respect to the previous element 
  -sticky -> positioned according to the scroll of the webpage
